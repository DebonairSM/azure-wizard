# Azure Wizard V2

A TypeScript Node.js application for discovering Azure configuration information and products through an interactive tree-based interface.

## Prerequisites

- Node.js 18+
- npm

## Installation

1. Navigate to the project directory:

```bash
cd azure-wizard-v2
```

1. Install dependencies:

```bash
npm install
```

1. Build the TypeScript code:

```bash
npm run build
```

## Running

Start the server:

```bash
npm start
```

Open your browser to `http://localhost:3030`
